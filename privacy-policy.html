Privacy Policy

Effective date: April 4, 2026

TL;DR — Agent Advisor is a fully local, open-source plugin. It does not collect, store, or transmit any personal data. Everything runs on your machine and stays on your machine.

1. About This Plugin

Agent Advisor is a free, open-source Claude Code plugin distributed under the MIT License. It provides a real-time dashboard for visualizing subagent activity and an AI-powered advisor for suggesting improvements.

2. Data Collection

We do not collect any data. Specifically:

  • No personal information is collected, stored, or processed
  • No usage analytics or telemetry are gathered
  • No cookies or local storage are used for tracking
  • No user accounts or authentication are required

3. How It Works

All data processed by Agent Advisor stays entirely on your local machine:

  • Hook events are sent to a local server on localhost:8099
  • Agent metrics and suggestions are stored locally in .claude/advisor-data/
  • The dashboard UI connects to the local server only via localhost
  • No data is transmitted to any external server, API, or third party

4. Third-Party Services

Agent Advisor does not integrate with, send data to, or receive data from any third-party services, advertising networks, or analytics platforms.

5. Network Communication

The only network communication occurs over localhost (your own machine) between the Claude Code hooks, the plugin's Node.js server, and the browser-based dashboard. No external network requests are made by the plugin.

6. Open Source

The complete source code is publicly available on GitHub. You can inspect, audit, and verify all of the above claims yourself. Contributions are welcome.

7. Children's Privacy

Since Agent Advisor does not collect any data from anyone, there are no special provisions regarding children's privacy. The plugin is safe for users of all ages.

8. Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated effective date. As an open-source project, all changes are tracked in the public repository.
